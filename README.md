# Super Markio

Author: Mark Bromley
Date: 20 March 2014


## Technical Details

Dependencies:
- pygame
- easypg 
- Python3 
- Tested in Ubuntu 12.

## Game Background

A fun implementation of an old classic with a few twists (Markio is a dinosuar. Live with it.).

Best played with a PS3 controller.
